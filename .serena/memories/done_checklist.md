После code change минимум: прочитать релевантные stable docs + impact, выполнить `dotnet build Final_Test_Hybrid.slnx`, затем `dotnet format analyzers --verify-no-changes` и `dotnet format style --verify-no-changes`. Для runtime-критичных изменений дополнительно нужен inspectcode warning/hint по изменённым .cs. Если изменён repo-tracked код или source-of-truth docs, требуется новый/обновлённый impact.
