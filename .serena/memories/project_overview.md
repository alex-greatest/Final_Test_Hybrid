Final_Test_Hybrid — критичная SCADA-система промышленных тестов на .NET 10 (WinForms + Blazor Hybrid, Radzen). Основной pipeline: Barcode -> PreExecutionCoordinator -> TestExecutionCoordinator -> 4 x ColumnExecutor. Есть runtime OPC UA, Modbus-диагностика, PostgreSQL/EF Core, SpringBoot/MES интеграция, Serilog/DualLogger. Документация и инварианты фиксируются в AGENTS.md и stable docs внутри Final_Test_Hybrid/Docs.
